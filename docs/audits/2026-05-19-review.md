## 🔴 Critique (à fixer cette semaine)

### 1. Leak Discord connections privées — vraie faille sécu
`GET /api/profile?uid=X` retourne `discordConnections[]` complet **sans filtrer par `visibleOnProfile`**. Le toggle est ignoré côté serveur — filtrage uniquement client (`app/profile/[id]/page.tsx:582`). Un visiteur peut récupérer tous les comptes "masqués" d'un joueur via curl.

**Fix** : filtrer dans `app/api/profile/route.ts` ligne ~130 — `discordConnections.filter(c => c.visibleOnProfile === true)`.

### 2. fontSize <12px généralisé — viole `feedback_lisibilite`
~200 occurrences de `fontSize: '9px'/'10px'/'11px'` malgré la règle. Pages affectées : `components/structure/CrossTeamTodosPanel.tsx`, `app/community/page.tsx`, `components/home/ConnectedDashboard.tsx`, `Sidebar.tsx`. Sur un site déjà jugé "trop petit", c'est le problème UX n°1.

**Fix** : sweep global, minimum 11px (compromis) ou 12px (cible).

### 3. Page `/competitions` = stub trompeur
Liste hardcodée de 2 entrées qui pointent vers le vieux site `springs-esport.vercel.app`. Titre "TOUTES LES COMPÉTITIONS" pour 2 redirects externes = mensonge user. Dupliquée 3× (`app/competitions/page.tsx:3-36`, `app/page.tsx:18-49`, `VisitorLanding.tsx:25-56`).

**Fix court terme** : message "Bientôt" honnête. **Fix long terme** : Phase 3.
